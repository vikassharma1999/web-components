# Getting Started with React Setup with Storybook

```
Steps To setup this project in your local system:
1.Clone this repo by using below command:
  git clone https://github.com/vikassharma1999/web-components.git
2.Go to inside web-components folder by using below command:
  cd web-components
3.Now type "npm install" in your terminal to install dependencies.
4.Now run "npm run storybook" to launch storybook server.
